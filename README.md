# Kondratieff Cycle: Quantitative Empirical Test

A reproducible quantitative test of the Chinese investment slogan **"人生发财靠康波"** (*"wealth comes from the Kondratieff wave"*), popularized by economist Zhou Jintao (周金涛, 1972–2016). This repository contains the data, code, and reproducible results for the **Myth Buster #02** article.

> **Companion article:** [Does Wealth Really Follow the Kondratieff Cycle? — A Five-Test Empirical Investigation](https://yifan70932.github.io/myth-busters/02-kangbo) *(replace with your actual hosted URL)*

## The hypothesis being tested

Zhou Jintao claimed in 2016 that human wealth-building opportunities follow a 60-year Kondratieff cycle, identifying three specific "wealth windows" in living memory:

| Window | Predicted opportunity | Cohort |
|---|---|---|
| **1999** | First window for those born ≤ 1959 | Pre-Boomers |
| **2008** | First window for those born 1960–1984 | Gen X |
| **2019** | First window for those born ≥ 1985 | Millennials |
| 2030 (extrapolated) | Next window | — |

His claim is empirically falsifiable: if Kondratieff windows really exist, buying major asset classes at the start of these years should produce returns that rank among the best, *not* among the worst.

## Five tests

| # | Test | Method | Result |
|---|---|---|---|
| 1 | **Event study** | 10-year CAGR rank vs all candidate years 1995–2014 | A-shares 2008 = rank 20/20 (worst); S&P 500 1999 = rank 20/20 (worst) |
| 2 | **Statistical inference** | Welch t-test, Mann-Whitney U, Bootstrap permutation (n=10,000) | All 12 p-values > 0.05 |
| 3 | **Risk-adjusted returns** | Sharpe ratio (rf = 3.5%) | Only 2 of 8 windows beat baseline |
| 4 | **Drawdown analysis** | Max drawdown over 10y | 6 of 8 windows have *worse* drawdown than baseline |
| 5 | **Spectral analysis** | Welch PSD on US GDP growth 1870–2024 | No significant peak at 60y (bootstrap p = 0.69) |

**Conclusion**: None of the five tests supports treating Kondratieff windows as actionable investment signals.

## Reproducing the results

```bash
# 1. Clone the repo
git clone https://github.com/yifan70932/Kondratieff-Cycle.git
cd Kondratieff-Cycle

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the four analysis scripts in order
cd src
python 01_event_study.py        # Tables 1 & 2
python 02_statistical_tests.py  # Table 3
python 03_risk_adjusted.py      # Tables 4 & 5
python 04_spectral_analysis.py  # Figure 2
```

Outputs are written to `../results/`. Total runtime: under 2 seconds on a standard laptop.

All bootstrap procedures use **random seed 42** (set in `utils.py`) for full reproducibility.

## Methodology notes

**CAGR**: 10-year compound annual growth rate from year-start price to year-start price 10 years later. Price returns only (excludes dividends and rental income); applied uniformly to all start years, so relative comparisons are unaffected.

**Sharpe ratio**: `(annualized mean return − risk-free rate) / annualized volatility`. Risk-free rate = 3.5% (long-run mean of 10-year US Treasury 1995–2024, FRED `DGS10`).

**Max drawdown**: largest peak-to-trough decline during the 10-year holding period.

**Spectral analysis**: linear detrending followed by Welch's method (`scipy.signal.welch`, `nperseg=60`, `scaling='density'`). Bootstrap significance via 5,000 random permutations of the detrended series.

**Bootstrap permutation test**: 10,000 iterations of pooled-shuffle resampling; reports two-sided p-value as the fraction of |Δ_boot| ≥ |Δ_observed|.

- **Bolt, J. & van Zanden, J. L. (2024)**. Maddison-style estimates of the evolution of the world economy. *Journal of Economic Surveys*, 39, 631–671. [doi:10.1111/joes.12618](https://doi.org/10.1111/joes.12618)

- **Bieshaar, H. & Kleinknecht, A. (1984)**. Kondratieff long waves in aggregate output? An econometric test. *Konjunkturpolitik*, 30(5), 279–303. *(Supports K-waves)*
- **Korotayev, A. V. & Tsirel, S. V. (2010)**. A spectral analysis of world GDP dynamics. *Structure and Dynamics*, 4(1). *(Supports K-waves)*

- **Metz, R. (2011)**. Do Kondratieff waves exist? *Cliometrica*, 5(3), 205–238. [doi:10.1007/s11698-010-0057-9](https://doi.org/10.1007/s11698-010-0057-9) *(Methodological survey)*

- **Efron, B. & Tibshirani, R. (1993)**. *An Introduction to the Bootstrap*. Chapman & Hall/CRC.

The literature is **divided** on whether long waves exist in macro data. This repository takes no firm position on that question; instead, it tests the much narrower (and individually verifiable) claim that *Zhou Jintao's three specific years (1999, 2008, 2019) are exceptional buy-points*. They are not.

## Caveats

- **Beijing housing pre-2008 data has measurement uncertainty** due to market segmentation between commodity, welfare, and resale housing.
- **2030 window cannot yet be tested** — it hasn't happened. But Zhou's framework predicted the world would be "firmly in the fifth Kondratieff depression" by 2025; observed 2024–2026 equity performance contradicts that prediction.
- **The 10-year horizon is one specific window length**; results are qualitatively similar at 5- and 15-year horizons (not shown in main analysis).
- **Counterfactual nature**: a typical Chinese investor in 1999 could not actually allocate all savings to Beijing housing (capital and quota barriers existed). The analysis tests Zhou's *theory*, not its practical executability.
